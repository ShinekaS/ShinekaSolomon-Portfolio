Here are four projects that I have worked on from March 2023 to May 2023. Two of the project are solo projects (Plant Life, and The Pet Place), and two project are team projects (Buck-It List, and Clean & Green).

https://plant-life.onrender.com

Plant Life is a React structured project done as a solo project for class. The goal was to return data from a Public API. This project is deployed on Render.com and may take a fw minutes to load.

https://shinekas.github.io/JSI-Project/

Buck-It List! was my first team project and my first project fetching from a Public API. I've worked in team enviroments many times before, but learning to work as a part of a team, taking coding different area, while learning and respecting other members coding styles was a very new experience. All three team members very pleased with this experience and the worked that we produced.

https://clean-and-green.onrender.com/

Clean & Green was done as a team project assignment for class. In doing this project we learned the value of User stories. The three members of the team worked as a unit to make decisions, plan, code and style. We reached out MVP goals with hopes to return to work on stretch goals.

https://shinekas.github.io/gfAPI/

The Pet Place project was a two part project. First we learned to build our own APIs. We then traded our APIs with other classmates with the instructions to build a front end around the other student's API. I received a very limited API but with it, I designed, The Pet Place, which in theory is a Pet rental service. Perhaps it would be used for terminally ill patients needing uplifting furry companionship.
